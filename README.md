# OOH Advertising Platform

A premium, state-of-the-art management suite for Out-of-Home (OOH) advertising campaigns.

## Features
- **Deployment Status**: Real-time monitoring of campaign activation across nodes.
- **Campaign Builder**: Inventory selection and budget review.
- **Asset Config**: Detailed configuration and creative management for digital billboards.
- **Campaign Approval**: Management feedback and approval portal.

## Technology Stack
- **Frontend**: React + Tailwind CSS (Vite)
- **Design System**: Material Design 3 inspired tokens
- **Aesthetics**: Glassmorphism, premium gradients, and smooth animations.

## How to Run
### Static Version (Preview)
Open any file in the `static/` directory in your browser:
- `static/deployment-status.html`
- `static/campaign-builder.html`
- `static/asset-configuration.html`
- `static/campaign-approval.html`

### Development Version (React)
1. Navigate to the root directory.
2. Run `npm install`.
3. Run `npm run dev`.

## Workspace Recommendation
Please set `/Users/mac/.gemini/antigravity/scratch/ooh` as your active workspace to explore the code.
